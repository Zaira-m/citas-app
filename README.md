#  Citas App – Unidad 2

Proyecto desarrollado en **Ionic + Angular** para la evaluación de la Unidad 2 de Programación Híbrida.  
La aplicación muestra citas con su autor y permite generar nuevas citas mediante un componente reutilizable.

---

##  Tecnologías
- Ionic  
- Angular  
- Typescript  
- Capacitor  

---

##  Herramientas de Desarrollo
- Visual Studio Code  
- Node.js  
- Git / GitHub  

---

##  Funcionalidades
- Mostrar una cita aleatoria  
- Componente reutilizable para visualizar citas  
- Uso de Inputs y Outputs para comunicación entre componentes  
- Servicio para manejar la lógica de las citas
